arctern.plot.choroplethmap¶

arctern.plot.choroplethmap(ax, region_boundaries, weights, bounding_box, color_gradient, color_bound=None, opacity=1.0, coordinate_system='EPSG:3857', aggregation_type='max', **extra_contextily_params)[源代码]¶

在 Matplotlib 中绘制轮廓图。

参数

  • ax (matplotlib.axes.Axes) – 绘制几何图形的画布的坐标轴。

  • region_boundaries (GeoSeries) – 一组作为要绘制的区域边界的多边形。

  • weights (Series) – 多边形的颜色权重。

  • bounding_box (list) – 地图的边界，如 [west, south, east, north]。

  • color_gradient (list) – 颜色梯度。你可以使用 [“hex_color”] 来为所有几何体指定相同的颜色，或者使用 [“hex_color1”, “hex_color2”] 来指定从 “hex_color1” 到 “hex_color2” 的渐变色。

  • color_bound (list, optional) – color_gradient 的权重范围 [w1, w2]，默认值为 0。只有在 color_gradient 有两个值 [“color1”, “color2”] 的情况下才需要指定 color_bound。将 “color1” 绑定到 w1，将 “color2” 绑定到 w2。当某个权重小于 w1 或大于 w2 时，此权重将相应地被截断为 w1 或 w2。

  • opacity (float, optional) – 多边形的不透明度，范围是 0.0 到 1.0，默认值为 1.0。

  • coordinate_system (str, optional) – 为所有几何图形设置 CRS（Coordinate Reference System，坐标参考系），默认值为 ‘EPSG:3857’。目前仅支持 SRID（Spatial Reference Identifier，空间参考标识符）作为 CRS 的 WKT 表示。

  • aggregation_type (str, optional) – 聚合类型，默认值为 ‘max’。

  • **extra_contextily_params (dict) – 传到 contextily.add_basemap 中的其他参数。
