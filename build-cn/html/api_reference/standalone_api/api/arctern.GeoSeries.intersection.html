arctern.GeoSeries.intersection¶

GeoSeries.intersection(other)[源代码]¶

对于 GeoSeries 中的每个几何体以及 other 中给出的对应几何体，计算它们的交集。

参数

other (geometry or GeoSeries) – 用于计算第一个 GeoSeries 中的几何体与其交集的几何体或 GeoSeries。* 如果 other 是几何体，那么此方法计算 GeoSeries 中的每个几何体与 other 的交集。* 如果 other 是一个 GeoSeries，则此方法计算 GeoSeries 中的每个几何体与 other 中具有相同索引的几何体的交集。

返回

GeoSeries 中的每个几何体与 other 中给出的对应几何体的交集。

返回类型

GeoSeries

实际案例

>>> from arctern import GeoSeries
>>> s1 = GeoSeries(["POLYGON ((1 1,1 2,2 2,2 1,1 1))"])
>>> s2 = GeoSeries(["POLYGON ((2 1,3 1,3 2,2 2,2 1))"])
>>> s2.intersection(s1)
0    LINESTRING (2 2,2 1)
dtype: GeoDtype
