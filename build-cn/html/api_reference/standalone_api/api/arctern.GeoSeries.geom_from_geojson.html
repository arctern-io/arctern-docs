arctern.GeoSeries.geom_from_geojson¶

classmethod GeoSeries.geom_from_geojson(json, crs=None)[source]¶

Constructs geometries from GeoJSON strings.

json is Series so that geometries can be created in batch.

Parameters

  • json (Series) -- String representations of geometries in JSON format.

  • crs (str, optional) -- A string representation of Coordinate Reference System (CRS). The string is made up of an authority code and a SRID (Spatial Reference Identifier), for example, "EPSG:4326".

Returns

Sequence of geometries.

Return type

GeoSeries

Examples

>>> from pandas import Series
>>> from arctern import GeoSeries
>>> json = Series(['{"type":"LineString","coordinates":[[1,2],[4,5],[7,8]]}'])
>>> GeoSeries.geom_from_geojson(json)
0    LINESTRING (1 2,4 5,7 8)
dtype: GeoDtype
