arctern.GeoSeries.to_file¶

GeoSeries.to_file(fp, mode='w', driver='ESRI Shapefile', **kwargs)[source]¶

Writes a GeoSeries to a file or OGR dataset.

Parameters

  • fp (str, pathlib.Path, or file-like object) -- A dataset resource identifier or file object.

  • mode (str) --

    Mode of writing data to file.

    • 'a': Append

    • 'w' (default): Write

    Not all driver support append, see the "supported drivers" below for more infomation.

  • driver (str) -- The OGR format driver, by default "ESRI Shapefile". It represents a translator for a specific format. See supported drivers for more information.

  • **kwargs --

  • to be passed to fiona.open. For example, layer or enabled_drivers. See `fiona.open <https (Parameters) --
