arctern.GeoSeries.convex_hull¶

property GeoSeries.convex_hull¶

对于 GeoSeries 中的每个几何体，返回包围它的最小凸几何体。

  • 对于多边形，返回的几何体是包围它的最小凸几何体。

  • 对于几何体集合，返回的几何体是将集合中的所有几何体包围起来的最小凸几何体。

  • 对于点或线，返回的几何体与原来的几何体相同。

返回

凸几何体的序列。

返回类型

GeoSeries

实际案例

对于 MULTIPOLYGON，MULTISURFACE 和 GEOMETRYCOLLECTION 等几何体集合，convex_hull 忽略点和线元素，并创建包围所有多边形元素的最小凸几何体。

首先，创建一个 MULTIPOLYGON 对象，该对象包含一个凹面多边形和一个矩形。

>>> import matplotlib.pyplot as plt
>>> from arctern import GeoSeries
>>> from arctern.plot import plot_geometry
>>> fig, ax = plt.subplots()
>>> g0 = GeoSeries(["MULTIPOLYGON(((0 0,0 2,1 1,2 2,2 0,0 0)),((2 0,2 2,3 2,3 0,2 0)))"])
>>> plot_geometry(ax,g0)

然后，使用 convex_hull 获得最小凸几何体，该凸几何将 MULTIPOLYGON 对象中的所有几何体包围起来。

>>> g1 = g0.convex_hull
>>> fig, ax = plt.subplots()
>>> plot_geometry(ax,g1)

让我们看看 convex_hull 如何处理包含半圆和矩形的 GEOMETRYCOLLECTION。

>>> fig, ax = plt.subplots()
>>> ax.axis('equal') 
>>> g4=GeoSeries(["GEOMETRYCOLLECTION(CURVEPOLYGON(CIRCULARSTRING(1 0,0 1,1 2,1 1,1 0)),polygon((1 0,1 2,2 2,2 0,1 0)))"])
>>> plot_geometry(ax,g4.curve_to_line())

使用 convex_hull 来获取最小凸几何体，该凸几何体将 GEOMETRYCOLLECTION 对象中的所有几何体包围起来。由于半圆形和矩形是凸形的，因此返回的凸几何体只是两个几何体的组合，并且与原来的几何体相同。

>>> fig, ax = plt.subplots()
>>> ax.axis('equal') 
>>> print(g4.convex_hull.to_wkt()[0])
CURVEPOLYGON (COMPOUNDCURVE (CIRCULARSTRING (1 0,0 1,1 2),(1 2,2 2,2 0,1 0)))
>>> plot_geometry(ax,g4.convex_hull.curve_to_line())

convex_hull 不会对 POINT，MULTIPOINT，LINESTRING，MULTILINESTRING 和 CIRCULARSTRING 进行任何更改。

下面的 GeoSeries s1 包含一个点、一条线和一个凸多边形。

>>> fig, ax = plt.subplots()
>>> ax.axis('equal') 
>>> s = GeoSeries(["POINT(2 0.5)", "LINESTRING(0 0,3 0.5)",  "POLYGON ((1 1,3 1,3 3,1 3, 1 1))"])
>>> plot_geometry(ax,s)

从 convex_hull 返回的几何体与原来的几何体完全相同。

>>> fig, ax = plt.subplots()
>>> ax.axis('equal') 
>>> print(s.convex_hull)
0                    POINT (2.0 0.5)
1           LINESTRING (0 0,3.0 0.5)
2    POLYGON ((1 1,1 3,3 3,3 1,1 1))
dtype: GeoDtype
>>> plot_geometry(ax,s.convex_hull)
