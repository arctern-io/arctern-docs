arctern.GeoSeries.to_wkt¶

GeoSeries.to_wkt()[源代码]¶

将 GeoSeries 中的所有几何体转换为 WKT 字符串。

返回

WKT 格式的几何体序列。

返回类型

Series

实际案例

>>> from arctern import GeoSeries
>>> s = GeoSeries(["POINT(1 1)"])
>>> s
0    POINT (1 1)
dtype: GeoDtype
>>> s.to_wkt()
0    POINT (1 1)
dtype: object
