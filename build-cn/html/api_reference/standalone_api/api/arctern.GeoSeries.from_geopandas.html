arctern.GeoSeries.from_geopandas¶

classmethod GeoSeries.from_geopandas(data)[source]¶

Constructs an arctern.GeoSeries object from a geopandas.GeoSeries object.

Parameters

data (geopandas.GeoSeries) -- A geopandas.GeoSeries object.

Returns

An arctern.GeoSeries object.

Return type

arctern.GeoSeries

Examples

>>> import geopandas as gpd
>>> from shapely.geometry import Point, Polygon
>>> from arctern import GeoSeries
>>> gpd_s = gpd.GeoSeries([Point(1,1), Polygon(((1,1), (1,2), (2,3), (1,1)))])
>>> arctern_s = GeoSeries.from_geopandas(gpd_s)
>>> arctern_s
0                    POINT (1 1)
1    POLYGON ((1 1,1 2,2 3,1 1))
dtype: GeoDtype
