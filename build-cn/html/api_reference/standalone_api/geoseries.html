GeoSeries¶

GeoSeries 继承于 pandas Series。它是一个存储和处理几何数据的 Series，并在内部将几何体存储为字节对象。

构造函数¶

GeoSeries([data, index, name, crs])

用于存储一组几何对象的一维 Series。

属性¶

GeoSeries.is_valid

检查 GeoSeries 中的每个几何体是否为有效格式，例如 WKT 和 WKB 格式。

GeoSeries.length

计算 GeoSeries 中每个几何体的长度。

GeoSeries.is_simple

检查 GeoSeries 中的每个几何体是否为“简单”的。

GeoSeries.area

计算 GeoSeries 中每个几何体的 2D 笛卡尔（平面）面积。

GeoSeries.geom_type

返回 GeoSeries 中每个几何体的类型。

GeoSeries.centroid

返回 GeoSeries 中每个几何体的中心。

GeoSeries.convex_hull

对于 GeoSeries 中的每个几何体，返回包围它的最小凸几何体。

GeoSeries.npoints

返回 GeoSeries 中每个几何体的点的数量。

GeoSeries.envelope

返回 GeoSeries 中每个几何体的最小边界矩形。

构造几何体¶

GeoSeries.point(x, y[, crs])

根据给定的坐标构造 POINT 对象。

GeoSeries.polygon_from_envelope(min_x, …)

在给定的空间范围内构造矩形 POLYGON 对象。

GeoSeries.geom_from_geojson(json[, crs])

根据 GeoJSON 字符串构造几何体。

GeoSeries.as_geojson()

将 GeoSeries 中的所有几何体转换为 GeoJSON 字符串。

GeoSeries.to_wkt()

将 GeoSeries 中的所有几何体转换为 WKT 字符串。

GeoSeries.to_wkb()

将 GeoSeries 中的所有几何体转换为 WKB 字符串。

GeoSeries.to_geopandas()

将 arctern.GeoSeries 对象转换为 geopandas.GeoSeries 对象。

GeoSeries.from_geopandas(data)

由 geopandas.GeoSeries 对象构造一个 arctern.GeoSeries 对象。

处理几何体¶

GeoSeries.buffer(distance)

对于每个几何体，将其所有点向远离几何体中心的方向移动以构造新的几何体。

GeoSeries.precision_reduce(precision)

对于 GeoSeries 中每个几何体的坐标，将有效数字的位数减少到给定的数字。

GeoSeries.intersection(other)

对于 GeoSeries 中的每个几何体以及 other 中给出的对应几何体，计算它们的交集。

GeoSeries.make_valid()

在不丢失任何输入顶点的情况下，为 GeoSeries 中的每个几何体创建有效的表示。

GeoSeries.simplify(tolerance)

返回使用 Douglas-Peucker 算法 计算的 GeoSeries 中每个几何体的简化版本。

GeoSeries.set_crs(crs)

为 GeoSeries 中的所有几何体设置 CRS（Coordinate Reference System，坐标参考系）。

GeoSeries.crs

返回 GeoSeries 的 CRS（Coordinate Reference System，坐标参考系）。

GeoSeries.to_crs(crs)

Transforms the Coordinate Reference System (CRS) of the GeoSeries to crs.

GeoSeries.curve_to_line()

Converts curves in each geometry to approximate linear representations.

空间关系¶

GeoSeries.geom_equals(other)

对于 GeoSeries 中的每个几何体以及 other 中给出的对应几何体，检查第一个几何体是否与另一个几何体等价。

GeoSeries.touches(other)

对于 GeoSeries 中的每个几何体以及 other 中给出的对应几何体，检测它们是否接触。

GeoSeries.overlaps(other)

对于 GeoSeries 中的每个几何体以及 other 中给出的对应几何体，检查第一个几何体是否与另一个几何体空间重叠。

GeoSeries.crosses(other)

对于 GeoSeries 中的每个几何体以及 other 中给出的对应几何体，检查第一个几何体是否与另一个几何体空间相交。

GeoSeries.contains(other)

对于 GeoSeries 中的每个几何体以及 other 中给出的对应的几何体，检测第一个几何体是否包含另一个几何体。

GeoSeries.intersects(other)

对于 GeoSeries 中的每个几何体以及 other 中给出的对应的几何体，检测它们是否相交。

GeoSeries.within(other)

对于 GeoSeries 中的每个几何体以及 other 中给出的对应的几何体，检测第一个几何体是否在另一个几何体内部。

测量¶

GeoSeries.distance_sphere(other)

对于 GeoSeries 中的每个几何体以及 other 中给出的对应几何体，计算它们之间的最小球面距离。

GeoSeries.distance(other)

对于 GeoSeries 中的每个几何体以及 other 中给出的对应几何体，计算它们之间的最小 2D 笛卡尔（平面）距离。

GeoSeries.hausdorff_distance(other)

对于 GeoSeries 中的每个几何体以及 other 中给出的对应几何体，计算它们之间的 Hausdorff 距离。

聚合¶

GeoSeries.unary_union()

返回一个几何体，该几何体表示 GeoSeries 中所有几何体的并集。

GeoSeries.envelope_aggr()

返回 GeoSeries 中所有几何体的并集的最小边界矩形。

Pandas 方法¶

GeoSeries.isna()

检查缺失值。

GeoSeries.notna()

检查存在的（非缺失）值。

GeoSeries.fillna([value, method, axis, …])

用 WKT 或 WKB 格式的几何体填充 NA 值。

GeoSeries.equals(other)

检查两个对象是否包含相同元素。
