arctern.GeoSeries.to_wkb¶

GeoSeries.to_wkb()[source]¶

Transforms all geometries in the GeoSeries to WKB strings.

Returns

Sequence of geometries in WKB format.

Return type

Series

Examples

>>> from arctern import GeoSeries
>>> import pandas as pd
>>> pd.set_option("max_colwidth", 1000)
>>> s = GeoSeries(["POINT(1 1)"])
>>> s.to_wkb()
0    b'\x01\x01\x00\x00\x00\x00\x00\x00\x00\x00\x00\xf0?\x00\x00\x00\x00\x00\x00\xf0?'
dtype: object
