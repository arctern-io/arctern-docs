GeoDataFrame¶

GeoDataFrame inherits pandas DataFrame. It is a DataFrame to store and process geometric data.

Constructor¶

GeoDataFrame([data, index, columns, dtype, ...])

A GeoDataFrame object is a pandas.DataFrame that has columns with geometry.

The Coordinate Reference System (CRS) of arctern.GeoDataFrame.

GeoDataFrame.set_geometry(col[, inplace, crs])
